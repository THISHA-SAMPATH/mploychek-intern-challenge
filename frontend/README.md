# MPloyChek Frontend

This Angular app is part of the MPloyChek full-stack challenge.

For complete setup instructions, demo credentials, API details, Docker usage, and development notes, see the root [README.md](../README.md).

## Common Commands

```bash
npm start
npm run build
npm test
```

`npm start` runs Angular with `proxy.conf.json`, so frontend calls to `/api` are forwarded to the backend during local development.
